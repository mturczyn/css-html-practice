By defining tabindex attribute, it is focusable (for example by navigating with tab), and gets visual effect when focused.

Tab index can be any number higher or equal to -1. Numbers 1 and above specify order by which focus will be received. Special are:

  • 0 - it will follow global order of focus,
  • -1 - it will not receive users navigation, and only accept programmatic (JavaScript) navigation.
