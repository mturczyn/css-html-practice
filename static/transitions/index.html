Foreword

When interacting with web page, many elements have state of some sort, for example dropdown list can be opened or closed, buttons can be focused or hovered.

By default, CSS switches between states instantly.

By using transitions, we can interpolate between the initial state and the target state of the element.

To specify which property should be interpolated, property transition-property should be specified. The easiest is to write transition-property: all to enable transitions for all properties.

Example

Hover over or focus buttons below to see transitions on first button and compare with second button.

There are also other properties to use, transition-duration to specify duration of transition, transition-timing-function, to specify the timing function (linear, cubic, etc., same as for animations), transition-delay, to delay the transition.
