Basics

Animations are defined using keyframes, which represent state of a control at a specific time. Basic example:

@keyframes bg-color-animation-kf {
from { background: rgba(255, 0, 0, 0); }
to { background: rgba(255, 0, 0, 1); }
}

Below animations define animation-direction to alternate in order to play the animation backwards, to achieve "loop" effect. Other values for animation-direction are normal, reverse, alernate-reverse.
