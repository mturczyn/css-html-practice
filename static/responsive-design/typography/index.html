Typography

Scaling text

It's good to use combination of relative unit like em, rem, ch and viewport size, like:

html { font-size: calc(0.75rem + 1.5vw); }

Clamping text

As per example above, the font size could grow or shrink without limits, so here we can make use of clamp function.

html { font-size: clamp(1rem, 0.75rem + 1.5vw, 2rem); }

Line width and line height

Experts on typography say that 66 characters on the line is ideal, and good is somewhere around 60-80 characters.

This means that relative units like ch, rem, em makes most sense, and it's good practice to do so.

So text containers should use rather relative units.

Same goes for line height, it's good practice to define it like: line-height: 1.5 - which is unitless value, but it makes this value relative, and in this case it's recommended.

Fonts

Recommended web font is Roboto font, which can be downloaded from internet.

We can specify @font-face in order to reference some other font and use it in font-family CSS property.

It can take some time to load font, so in order to display default font we can specify CSS property font-display: swap;.
