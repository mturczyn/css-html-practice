Theming

This page is about theming. This site uses @media (prefers-color-scheme: dark) in order to apply different CSS rules to elements in this page, when user has dark mode.

We can also use picture element in order to swap images entirely based on user's theme.

Below image uses following HTML structure:

<picture>
<source srcset="..." media="(prefers-color-scheme: dark)" />
<img src="..." alt="..." />
</picture>

Forms

For forms, browser also provides default styling. We need to let the browser know that site offers more themes (for example: dark and light).

In order to add supported color theme, to let the browser know, we can do it in two ways:

  • through CSS: html { color-scheme: light; }
  • through meta element: <meta name="supported-color-schemes" content="light dark">

Also accent-color should be specified, in order to provide more style to checkboxes and some other form fields.
