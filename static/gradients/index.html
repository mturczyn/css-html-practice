Linear gradients

Also, direction of gradient can be defined, just use angle as first parameter, or to right (other options are left, top, bottom), to right is equivalent to 90deg angle.

We can also change where the transitions happens in the gradient with linear-gradient(black 50%, white) (percentage specifies the color stop).

We can specify as many colors and color stops in a linear gradient as we want.

Radial gradients

To create a gradient that radiates in a circular fashion, the radial-gradient() function steps in to help. It's similar to linear-gradient(), but instead of specifying an angle, you optionally specify a position and ending shape. If you just specify colors, the radial-gradient() will auto-select the position as center and select either a circle or ellipse, depending on the size of the box.

With different center and ellipse shape.

Conic gradient

There's another type of gradient conic-gradient.

Similairly to radial and linear gradient, we can adjust some things in conic gradient as well: center and angle, for example conic-gradient(from 45deg at 20% 30%, white, black).

repeating-linear-gradient

We can also repeat the gradient, if we specify the color stops and the gradient ends without filling the box. If the gradient doesn't appear to be repeating, probably it does not have set a length for one of the color stops.

Multiple gradients

You can also mix gradient functions on background properties, as well as defining as many gradients as you like, just like you would with a background image. For example, you can mix multiple linear-gradients together, or two linear-gradients with a radial gradient.
