:focus-within

Element containing this paragraph and below button has focus-within pseudo class used to style the parent when any of its children receives focus. For example focus button in order to see how whole element (so this paragraph and the button) gets bold font.

:focus-visible

Focus is received when clicking the element (eg. button) as well, so this property is used to show fucused style only when "truly" focused, with, for example, tab button.
